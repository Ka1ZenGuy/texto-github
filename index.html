    Introduccion

    Git es un software que nos permite mantener control de los cambios que se realizan sobre el código fuente de un proyecto.

    Git es un sistema de control de versiones FOSS (Free Open Source Software) y se utiliza en todo tipo de proyectos tanto profesionales como amateurs
